## 🔽 Como baixar o projeto

<br>

```bash
# Clonar repositório
$ git clone https://github.com/peedrojs/landing-page-react

# Entrar na pasta do projeto
$ cd landing-page-react

# Instalar dependências
$ yarn install

# Iniciar projeto
$ yarn start
```
